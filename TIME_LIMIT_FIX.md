# 时长限制倒计时修复 - 测试指南

## 修复内容

### 1. 倒计时到期自动退出
- 当倒计时到0时，现在会：
  1. 强制返回桌面（退出应用）
  2. 显示"使用时间已到"对话框
  3. 取消倒计时通知

### 2. 提高通知可见性
- 通知渠道重要性从 DEFAULT 提升到 HIGH
- 通知优先级从 DEFAULT 提升到 HIGH
- 添加了 VISIBILITY_PUBLIC 确保在锁屏上也显示
- 添加了详细的日志输出

### 3. 新增"时间已到"对话框
- 当时间用完时，会显示一个对话框提示用户
- 对话框会自动在3秒后关闭
- 用户也可以点击"我知道了"按钮关闭

## 测试步骤

### 测试1: 时长限制倒计时

1. **清除旧数据**
   - 打开应用 -> 设置 -> 调试工具 -> 清除今日记录

2. **创建测试策略**
   - 进入"专注"页面
   - 点击"+"创建新策略
   - 选择1个测试应用（例如：浏览器）
   - 设置使用时长限制：
     - 选择"每个应用"模式
     - 设置为 0小时 2分钟（用于快速测试）
   - 不要设置时间范围限制和次数限制
   - 激活策略

3. **测试倒计时**
   - 打开被限制的应用
   - **检查点1**: 下拉通知栏，应该看到一个通知显示"剩余时间: 2分钟0秒"
   - **检查点2**: 通知应该每秒更新，显示倒计时（1分59秒, 1分58秒...）
   - **检查点3**: 等待2分钟，当倒计时到0时：
     - 应用应该自动退出（返回桌面）
     - 应该显示"使用时间已到"对话框
     - 通知应该消失

4. **验证限制生效**
   - 尝试再次打开被限制的应用
   - 应该立即被阻止，显示"使用时长已达上限"对话框

### 测试2: 通知显示问题排查

如果通知没有显示，请检查：

1. **通知权限**
   - 设置 -> 应用 -> UMind -> 通知 -> 确保已开启

2. **通知渠道设置**
   - 设置 -> 应用 -> UMind -> 通知 -> 应用使用提醒 -> 确保重要性为"高"

3. **查看日志**
   ```bash
   ./debug_logs.sh
   ```
   搜索以下关键词：
   - "showUsageNotification" - 确认通知方法被调用
   - "Notification built" - 确认通知被创建
   - "Starting countdown" - 确认倒计时启动
   - "TIME'S UP" - 确认时间到期

4. **勿扰模式**
   - 确保手机没有开启勿扰模式
   - 确保通知不会被系统屏蔽

### 测试3: 长时间使用测试

如果想测试更长的时间限制：

1. 设置时长限制为 0小时 30分钟
2. 打开被限制的应用
3. 确认通知显示"剩余时间: 30分钟0秒"
4. 使用应用一段时间（例如5分钟）
5. 检查通知是否更新为"剩余时间: 25分钟XX秒"
6. 继续使用直到时间用完

## 预期行为

### 正常流程
1. 打开应用 -> 显示倒计时通知
2. 使用应用 -> 通知每秒更新倒计时
3. 时间到 -> 自动退出应用 + 显示对话框
4. 再次打开 -> 立即被阻止

### 通知内容
- 标题: "应用名称 使用提醒"
- 内容: "⏱️ 剩余时间: X分钟Y秒"
- 如果同时有次数限制: "⏱️ 剩余时间: X分钟Y秒\n🔢 剩余次数: Z次"

### 对话框内容
- 标题: "⏰ 使用时间已到"
- 内容: "应用「XXX」的使用时间已用完\n\n请注意休息，保持专注！"

## 常见问题

### Q1: 通知不显示
**A**:
1. 检查通知权限是否开启
2. 检查通知渠道重要性是否为"高"
3. 查看日志确认通知是否被创建
4. 重启应用和无障碍服务

### Q2: 倒计时不准确
**A**:
倒计时是基于打开应用时的剩余时间。如果之前使用过应用，剩余时间会相应减少。使用"清除今日记录"功能重置。

### Q3: 时间到了但没有退出应用
**A**:
1. 查看日志确认是否执行了 "TIME'S UP" 和 "Forced home action"
2. 检查无障碍服务是否正常运行
3. 某些系统应用可能无法被强制退出

### Q4: 退出应用后立即又能打开
**A**:
这是bug，说明使用时间没有正确记录到数据库。查看日志中的 "Recording usage" 和 "getUsageDuration" 确认。

## 调试日志关键词

运行 `./debug_logs.sh` 并搜索：

- `showUsageNotification` - 通知显示
- `Starting countdown` - 倒计时启动
- `TIME'S UP` - 时间到期
- `Forced home action` - 强制退出
- `showTimeUpDialog` - 显示时间到期对话框
- `Recording usage` - 记录使用时长
- `getUsageDuration` - 获取已使用时长
